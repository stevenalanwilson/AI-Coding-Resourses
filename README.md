## Contributing

Have a resource that should be included? Contributions are welcome!

### How to Contribute

1. **Fork this repository**
2. **Add your resource** to the appropriate section with:
   - A descriptive title and link
   - A brief label (e.g., *Official guide*, *Community resource*, *Tutorial*)
   - A 1-2 sentence description explaining what it covers and who it's for
3. **Submit a pull request** with a clear description of the resource you're adding

### Contribution Guidelines

- Resources should be relevant to Claude AI, Claude Code, or AI-assisted development
- Links must be active and point to quality content
- Descriptions should be clear, concise, and helpful
- Prefer official documentation and well-maintained community resources
- Keep the focus on practical, actionable content
